# 🚗 CarJudo - TCO Intelligence

CarJudo helps buyers understand the REAL 5-year cost of owning a used vehicle — not just the sticker price.

## Features
- Budget-first vehicle prediction
- ML-based year & mileage estimation
- 5-year Total Cost of Ownership breakdown
- What-if scenario modeling

## Tech Stack
- Python
- Streamlit
- Scikit-learn
- Pandas
